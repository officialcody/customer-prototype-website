About the Program
We have some new robot technology that can check oil and gas pipes for damage and decay from the inside. We've been testing this technology for a while, and it looks feasible. The robots crawl inside the pipes at about .1 (one-tenth) KM/hour, and the robots have a Mean Time Between Failure (MTBF) rate of about 1,000KM.
  1. The user is presented with a map of key oil/gas sites within India.
    • The map allows the user to show a street view map with country borders.
    • The map allows the user to zoom in and out.
    • The map will display icons indicating the locations of 3 key oil sites (Hyderabad-Vijayawada-Vizag).
  2. When the user moves the mouse over a site marker, an info window is to appear above the site that contains the following site information:
    • Site Name
    • Site type (e.g. oil, gas or oil and gas)
    • Location (latitude and longitude)
    • For the given sites (Hyderabad, Vijayawada and Vizag), the marker is a link to a webpage about the site. When the marker is clicked upon by the user a new web page is opened that provides a picture of the site.
